# AI-Powered Ecommerce Back-End

This is the **backend** for the AI-Powered Ecommerce application, built with **Node.js, Express, and MongoDB**.  
It provides secure APIs for **product management, user authentication, and order handling**, ready to integrate with the front-end.

## Features
- RESTful API endpoints for products, categories, users, and orders
- User authentication & authorization with **JWT**
- CRUD operations for products and categories
- CORS enabled for secure cross-origin requests
- Environment variable configuration with **dotenv**
- Fully structured for integration with React front-end

## Tech Stack
- **Node.js** – Server runtime environment
- **Express.js** – Web framework for building APIs
- **MongoDB** – NoSQL database for storing application data
- **JWT** – Authentication and authorization
- **CORS** – Cross-origin request handling
- **dotenv** – Environment variable management
